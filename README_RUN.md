AI Plant Doctor - Full project (frontend + backend)


Model is work proper on the the Plant village Dataset Images with the accuracy of 90+

1) For runing backend :

cd backend
pip install -r requirements.txt
python app.py

// http://127.0.0.1:5000 backend is runnig on this IP

for using gemini api create a .env file in backend folder and paste the api og gemini ai within like the formate is given below 
 
 ---------GEMINI_API_KEY = "Paste Your API KEY Here"------------


2) Frontend (new terminal):

cd frontend
npm install
npm run dev

// http://localhost:8000 frontend is runnig on this IP
